Project Overview

Atlantis is a web application developed by Team 12 for EECE/CS-3093C Software Engineering at the University of Cincinnati. The project provides a cloud-based platform built with Node.js, HTML5, and Bootstrap, and deployed on Microsoft Azure App Services.

Goals

  • Design and implement a functional full-stack web application
  • Practice Agile/Scrum development across multiple sprints
  • Deploy and maintain the application on Azure cloud infrastructure

Technologies

  • Node.js & JavaScript
  • HTML5 & Bootstrap
  • Azure App Services
  • GitHub Projects (Scrum board)
